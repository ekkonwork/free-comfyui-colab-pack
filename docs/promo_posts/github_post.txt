Free ComfyUI Colab Pack (popular models, GGUF-first, T4-friendly)

Body:
I published a repository with free ComfyUI Colab notebooks for popular models.

Highlights:
- Auto quant selection by VRAM budget
- GGUF-first model setup
- Match-only Lenovo UltraReal LoRA download logic
- Per-model workflow JSON files included
- Low-VRAM defaults for Colab Free T4
- Token prompts for Hugging Face and Civitai
