EN:
I also maintain a standalone project: Qwen3 VL AutoTagger CLI.
It generates stock-style title + keywords and can embed XMP metadata directly into output images.

Highlights:
- works without ComfyUI
- single image or batch folder processing
- optional XMP embedding via exiftool
- metadata JSONL export for pipelines
- optional 4-bit mode on CUDA
- Colab notebook included

If you work with stock pipelines or large image batches, this tool can save real time.
Stars, issues, and feedback are welcome.
If you want to support development, even $1 helps and goes to GPU costs and food.

Repo:
https://github.com/ekkonwork/qwen3-vl-autotagger-cli

Также поддерживаю отдельный проект: Qwen3 VL AutoTagger CLI.

- есть готовый Colab ноутбук
